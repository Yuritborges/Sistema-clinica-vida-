# 🏥 Sistema Clínica Vida

> Projeto acadêmico — Sistema de gestão para clínica médica. Primeiro sistema web desenvolvido de forma independente.

---

## 📋 Sobre o Projeto

Desenvolvido como projeto da faculdade de **Análise e Desenvolvimento de Sistemas**, este foi o primeiro sistema web construído de forma completamente independente. O projeto simula um sistema de gestão para uma clínica médica.

---

## ✨ Funcionalidades

- Cadastro de pacientes
- Agendamento de consultas
- Gestão básica da clínica

## 🎓 Contexto Acadêmico

Projeto desenvolvido durante o curso de **Análise e Desenvolvimento de Sistemas** como exercício prático de desenvolvimento web. Representa o início da jornada de desenvolvimento, evoluindo posteriormente para sistemas reais como o **Top Mall Utilidades**.

---

> *"Todo especialista já foi um iniciante"* — Este projeto marca o início de uma jornada que levou ao desenvolvimento de sistemas reais para clientes.
